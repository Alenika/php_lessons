Шпаргалка

Переход по директориям:

pwd — вывести текущую рабочую директорию

cd <путь> — перейти в директорию

Работа с файлами:

touch <filename> — создать пустой файл

mkdir <dir_name> — создать директорию

mkdir -p <dir_path> — создать родительские директории (если их нет)

rm <filename> — удалить файл

rm -r <dir_name> — удалить директорию с содержимым

cp <source> <destination> — копировать файл

mv <source> <destination> — переместить файл

Просмотр содержимого:

ls — показать содержимое текущей директории

ls -a — показать все файлы (включая скрытые)

cat <filename> — вывести содержимое файла

Работа с Git:

git status — проверить статус изменений

git add <file> — добавить файл в индекс

git commit -m "<message>" — сделать commit с сообщением

git push — отправить изменения на GitHub

git pull — загрузить изменения с GitHub

Работа с процессами:

ps aux — показать список всех процессов

kill <PID> — завершить процесс по ID
